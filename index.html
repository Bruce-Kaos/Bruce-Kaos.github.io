Hexic

Lifesteal SMP

Welcome to the Hexic Lifesteal SMP website! The Hexic server is a minecraft server where, as the name sugests, you steal other player's hearts when you kill them. This server is split into seasons, after each season the world is restarted, and new players are invited. However new players can join at any time, but be aware, others will likely be ahead of you and see your hearts as easy pickings. See more about the server under the info tab.



Remember: Hacking is NOT allowed and you will be banned if caught. Things like Onix Client are allowed as they are not consided hacking. OP commands are also bannable. Teaming is not only allowed, but recommended. However, you should expect to be betrayed. Report glitches and hacking under contact.
